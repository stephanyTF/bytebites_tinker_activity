
# TF SUMMARY

The core concepts students need for Week 3 Tinker: ByteBits are how to use AI as a design tool to streamline app development. 

Some students may struggle to interpret the core features from the specs, especially since the instructions mention four classes but the project realistically requires five. It’s also easy for students to overrely on AI and accidentally skip steps, as I did when VS Code Chat generated methods before I completed the scaffolding. While AI was helpful in catching mistakes in my test script, it sometimes gave misleading confirmations or wrote test cases that didn’t actually test the intended behavior. 

After this experience, I would guide students to slow down, read confusing sections line by line, and confirm their understanding before asking AI for help. Taking time to think through the logic is a crucial and non skippable process for building a robust app.
